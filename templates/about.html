Snaptalk

Snaptalk is an AI-powered Image Caption Generator designed to bridge the gap between visual content and natural language. The system leverages Convolutional Neural Networks (CNN) to extract deep visual features from images and Long Short-Term Memory (LSTM) networks to generate meaningful, human-like captions.

This project demonstrates the practical application of Deep Learning in computer vision and natural language processing, enabling automatic image understanding and caption generation with high accuracy.
